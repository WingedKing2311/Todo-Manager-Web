# 📝 Task Manager Web App

A clean and interactive task management web application built using **HTML, CSS, and JavaScript**, supporting task organization with priority levels, due dates, and persistent storage.

---

## 🚀 Features

- ➕ Add tasks with priority and due date  
- ✏️ Edit existing tasks  
- ❌ Delete tasks  
- ✔️ Mark tasks as completed  
- 🔍 Filter tasks (All / Completed / Pending)  
- 📅 Sort tasks by due date  
- 💾 Persistent storage using localStorage  
- 🎨 Clean and responsive UI  

---

## 🛠 Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Storage:** Browser localStorage

## ⚙️ How It Works

- Tasks are stored as JavaScript objects  
- All tasks are saved in **localStorage** for persistence  
- Dynamic DOM updates handle UI rendering  
- Filtering and sorting are implemented using array operations
